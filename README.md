【開発者向け】
開発環境起動　：「npm run dev」
build　　　　：「npm run build」
Lintチェック　：「npm run lint」
開発DBのテーブル初期化とテーブル定義　　　　　：「npm run init-db」
開発DBのダミーデータ初期化とダミーデータ定義　：「npm run seed-db」
URL　　　　　　：「http://localhost:3000」
